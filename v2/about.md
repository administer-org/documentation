# Administer API V2

The Administer API is a rich collection of utilities for creating apps on Administer, the most powerful & open admin panel for Roblox. If you're looking to learn more about Administer, set it up, or make an app, you've come to the right place.

**Apps**: Learn how to use the [InvocationAPI](/v2/apps/types/invocation-api.md), [RichConfig](/v2/apps/types/richconfig.md), and [Bootstrap](/v2/server/apps.md) to make an Administer app using the [Creation Guide](/v2/apps/types/module.md)
**Setup Guide**: Get going quickly with our [setup guide](/quickstart/installation.md) for first-time Administer users.
**AOS Setup**: Developers can get their own AOS instance up within 5 minutes [here.](/AOS/maintaining/setup.md)
