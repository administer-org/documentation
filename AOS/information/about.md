# About AOS

## What is AOS?

AOS, or Administer Online Services, is what the Administer Marketplace uses as its backend. Administer also uses it for many other things, such as version checking, reports, feedback, and many other functions inside of Apps in the forms of AOS Plugins.

## Can I run my own server?

Yes! We encourage owning AOS servers for everybody to use. Getting an AOS Node up is very easy and lightweight. For more information, see our [setup guide.](/AOS/maintaining/setup)

## I want to report a bad server!

Administer Software does not control 3rd party servers. If a 3rd party server's API is being abused, you need to get in contact with its maintainer(s).
