# Official AOS Nodes

## What is an AOS node?

An AOS node is a server running AOS using the official database with other official plugins running on it. They are able to be connected to from Administer and the Administer Web Frontend. Official AOS nodes contain some web plugins which are useful for us but may not offer any value for you on your own server.

## Can I host my own?

Yes! See the [Installation](/AOS/maintaining/setup) guide.

## On my own instance, can I use the production database?

No, the production database is private and may not be accessed by your own server.

# AOS Database Cluster

Currently, the AOS Database runs on a cluster of 3 nodes, on all of our US-based servers. We will be adding a node in Europe to the cluster eventually. However, it is not possible yet. For the sake of security we do not give database addresses. For more information on our data security policies, please see the [privacy policy.](/AOS/legal/privacy)
