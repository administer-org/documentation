# Translation Keys

Administer's frontend supports translations with our custom locale system.

If you know a language that is not supported currently or would like to improve an existing one, you may log into the Administer Translator Portal at https://translate.admsoftware.org and submit changes.

## What translation keys can I use?

Please refer to our base language (en-US) here: https://github.com/administer-org/administer/blob/main/locales/en.json

## Can I add new ones?

Yes, we **sometimes** allow contributions adding new keys, usually as a part of larger features. We reserve the right to decline pull requests adding new locale keys if they are not a part of larger changes. These changes must be created on the root GitHub.

If you are looking to contribute a new language, improvements to an existing one, or adding keys which exist in English but not that language yet, please feel free to do so freely on the translator portal.
