# What is Administer?

Administer is an admin framework which takes modularity to a level you've never seen before. You only keep what you need.<br>
The framework allows for a library of apps with various features which you can add or remove as you so wish, you can even make your own!<br>
Our modern admin panel allows you to do virtually anything you'll ever need to.<br>
